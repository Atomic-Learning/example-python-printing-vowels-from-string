In this example, we will write a piece of Python code which prints all the vowels in a string.

input_string = "A longish string" for letter in input_string: if letter in "aeiouAEIOU": print(letter)

When this code runs, letter takes on each character in input_string in turn, and the if statement checks if that character is a vowel. It does this by seeing if the character appears in the string "aeiouAEIOU". Strings are case-sensitive, so both uppercase and lowercase vowels are included in the string to be checked against to make sure both uppercase and lowercase vowels from the original string are printed.

If the character currently being considered appears in the string of vowels, it is printed.
